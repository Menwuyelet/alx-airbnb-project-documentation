# 🏡 Airbnb Clone Backend — Core Functionalities

## ✨ Overview

This document outlines the core functionalities and main use cases for the Airbnb Clone backend system. It is designed to guide feature development and provide clarity on user interactions and data flows.

---

## 👤 Guest Functionalities

- **Account Management**
  - Register and manage personal profile.
  - Update contact information and preferences.

- **Searching and Booking**
  - Search for properties based on location, availability, and filters.
  - Book selected properties.

- **Payment Processing**
  - Securely pay for bookings using integrated payment methods.

- **Reviews**
  - Leave reviews and ratings for properties after a stay.

---

## 🏠 Host Functionalities

- **Account Management**
  - Create and maintain host profile.
  - Manage personal and contact information.

- **Property Management**
  - Add, edit, and remove property listings.
  - Set availability, pricing, and descriptions.

- **Booking Management**
  - View, accept, or cancel guest bookings.
  - Track booking statuses.

- **Payment Processing**
  - Receive payments for confirmed bookings.

- **Reviews**
  - View and respond to guest reviews.

---

## 🛡️ Admin Functionalities

- **System Management**
  - Manage user accounts and permissions.
  - Monitor property listings and bookings.
  - Oversee platform-wide operations and resolve disputes.
